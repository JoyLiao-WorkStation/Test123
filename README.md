# App Monitoring Guide
## English Instructions
- After launching the app, connect the device to your phone, tablet, or box via USB. When the system permission dialog pops up, tap "Allow" to grant access to the device. Once permission is granted, the "Start Monitoring" button will become enabled.
- Tap the "Start Monitoring" button to begin monitoring. The app will start detecting targets and display them as points on the radar chart.
- To stop monitoring, tap the "Stop Monitoring" button. If the button occasionally does not respond, you can unplug the USB device to forcibly terminate monitoring.
- If the USB device cannot be detected or fails to initialize, unplug the USB device and plug it in again to resolve the issue.
